Criterios de Evaluación

  1. Justificación: El proyecto responde a una necesidad real dentro del colegio, fomentando la conciencia ambiental y el aprendizaje práctico.
  2. Identificación del Problema: Se plantea la necesidad de implementar una huerta escolar que mejore el entorno y sirva como recurso pedagógico.
  3. Posibles Soluciones: Se proponen alternativas con distintos tipos de sensores y sistemas de riego automatizado.
  4. Diseño y Conceptualización: Se presenta un diagrama de bloques que muestra la interacción entre sensores, microcontrolador y aplicación web.
  5. Fuentes de Información: Se incluyen referencias confiables sobre los sensores utilizados.
  6. Presentación del Proyecto: Se organiza en una página web publicada en GitHub Pages, con repositorio accesible al público.
